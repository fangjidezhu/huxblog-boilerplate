## 开博初衷

##### 为什么选择GitHub
	在使用了各种博客平台后，总觉得定制性太差，而且无法离线编辑，这方面，GitHub能够免费托管项目且提供静态页面的托管，方便搭建一个易于发布的内容源，这是选择github的原因。当然现在国内的厂商比如`coding`和`csdn`也开拓了代码托管服务，考虑到用户量和活跃度，最终决定在GitHub开始写博，期待一段美妙的分享之旅。
